UT00 - Instalar Java en Windows 10

Resumen del Tema

En este tema se aborda el proceso de instalación de Java en un sistema operativo Windows 10. Se proporcionan instrucciones detalladas sobre cómo verificar si Java está instalado, dónde obtener la versión adecuada de Java desde la página oficial de Eclipse Temurin, y cómo seguir los pasos de instalación.

Es importante destacar que se recomienda elegir una versión LTS (Long-Term Support) para garantizar la estabilidad y el soporte a largo plazo. Además, se hace hincapié en la importancia de configurar la variable de entorno JAVA_HOME durante la instalación para poder ejecutar todos los comandos de Java, incluido javadoc, desde el terminal.

Una vez completada la instalación, se indica cómo verificar que Java se ha instalado correctamente y cómo comprobar que se puede ejecutar el comando javadoc.
